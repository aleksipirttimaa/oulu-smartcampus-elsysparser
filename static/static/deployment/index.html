Map with explanations of the location codes

The location identifier is in format RR-XXYY, where RR is room suffix, XX is column and YY is row.

You'll find that there are one set of columns from 01 to 29, but multiple sets of rows differentiated by the room.

For example: SM-1601 is Staff meeting room, column 16 row 01.

You can use the search field to filter sensors by any field. For example useful searches are: 'never' and 'BK-2602'
